Listas

Una lista puede contener varios elementos

["manzana","platano","naranja"]

CARACTERISTICAS:

  • Estan ordenados y empiezan indice 0
  • Se pueden cambiar, a単adir y modificar elementos de la lista
  • Permite elementos duplicados
  • Longitud de la lista: len(nombreLista)
  • Se pueden mezclar tipos de datos
  • La funcion type() devuelve: class 'list'

Tuplas

Una tupla puede contener varios elementos

("manzana","platano","naranja")

CARACTERISTICAS:

  • Estan ordenados y empiezan indice 0
  • No se pueden modificar, ni a単adir ni borrar
  • Permite elementos duplicados
  • Longitud de la lista: len(nombreTupla)
  • Se pueden mezclar tipos de datos
  • La funcion type() devuelve: class 'tuple'
  • Mas ligero que una lista
  • La tupla se crea a partir de un elemento y una coma: myTuple = ("hola",)

Set

Un Set puede contener varios elementos

{"manzana","platano","naranja"}

CARACTERISTICAS:

  • No estan ordenados
  • No se pueden modificar, pero SI a単adir y SI borrar
  • NO permite elementos duplicados
  • True y 1 SON LO MISMO
  • Se pueden mezclar tipos de datos
  • Longitud de la lista: len(nombreSet)
  • La funcion type() devuelve: class 'set'
  • Se puede comprobar si existe en el set print("banana" in mySet) #true

Diccionarios

Almacen de informacion en formato clave:valor

{"clave1":"valor1", "clave2":"valor2"}

CARACTERISTICAS:

  • Estan ordenados
  • Las claves son STR, los valores de cualquier tipo
  • Se pueden modificar, a単adir y borrar elementos
  • No estan permitidas las claves duplicadas !
  • Se pueden mezclar tipos de datos en los valores
  • La funcion type() devuelve: class 'dict'
  • Longitud de la lista: len(nombreDict)
  • Desempaquetado: for x, y in myDict.items(): print(x,y) clave-valor
